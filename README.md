## Project Description

This project demonstrates a simple Automated Machine Learning (AutoML) framework developed using Python and Scikit-learn.

The framework automatically loads a dataset, preprocesses the data, trains multiple machine learning algorithms, compares their performance, and selects the best-performing model based on accuracy.
